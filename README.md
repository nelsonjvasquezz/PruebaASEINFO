# Prueba ASEINFO
Prueba para empresa ASEINFO

#Instalación
Debe clonar el proyecto en su máquina local. Luego debe ejecutar el script de la base de datos
en SQL Server, ya que se ha trabajado bajo el método Data First.

#Funcionalidad del sistema
Se creó un Login y registro de usuarios y clientes, con confirmación de registro por medio de correo 
electrónico, se pueden agregar productos, se puede recuperar contraseña por medio de correo
